---
description: Deploy Next.js application to localhost
---

1. Install project dependencies
   ```bash
   npm install
   ```
   // turbo (auto-run safe)

2. Run the development server
   ```bash
   npm run dev
   ```
   // turbo (auto-run safe)

3. Open your browser and navigate to `http://localhost:3000` to view the site.

4. (Optional) If you need to specify a different port, set the `PORT` environment variable before running the dev server:
   ```bash
   PORT=4000 npm run dev
   ```
   Then visit `http://localhost:4000`.

5. To stop the server, press `Ctrl+C` in the terminal.
